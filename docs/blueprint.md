# **App Name**: BingeTime

## Core Features:

- Search and Add Media: Users can search for movies and TV shows using an autocomplete search bar connected to the TMDB API. Upon selection, the movie/show is added to their watch list, and its duration is fetched and stored.
- Total Watch Time Calculation: The app calculates the total watch time based on the items in the user's watch list and displays it in DD:HH:MM:SS format.
- User Authentication: Users can log in using their Google account via Firebase Authentication. A guest mode is also available with local storage fallback.

## Style Guidelines:

- Primary color: Dark blue (#1A202C) for a sleek and modern feel.
- Secondary color: Light gray (#EDF2F7) for backgrounds and subtle contrasts.
- Accent: Teal (#4DC0B5) for interactive elements and highlights.
- Clean and readable sans-serif font for body text and titles.
- Use minimalist and consistent icons for navigation and actions.
- Card-based layout for displaying movies/shows in the watchlist.
- Smooth transitions and subtle animations for user interactions.
